# ContactHub

[Live Demo](https://shahdz1.github.io/contactHub/)

ContactHub is a simple and responsive contact management website built using **HTML, CSS, Bootstrap, and JavaScript**. It allows users to manage and organize their contacts easily with a clean and modern interface.

## Features
- Add, edit, and delete contacts
- Responsive design for desktop and mobile
- Built with Bootstrap for a modern layout
- Interactive UI with JavaScript

## Tech Stack
- **HTML**: Structure of the website
- **CSS**: Styling and layout
- **Bootstrap**: Responsive design and components
- **JavaScript**: Functionality and interactivity

## How to Use
1. Open the [Live Demo](https://shahdz1.github.io/contactHub/) in your browser.
2. Start adding your contacts.
3. Edit or delete contacts as needed.
